Databricks Serverless GPU CLI wheel packages for sharing and version tracking.

See [sgcli_wheel/README.md](sgcli_wheel/README.md) for the full release table and changelog.

## Quick Install

```bash
# Latest stable
pip install sgcli_wheel/databricks_serverless_gpu_cli-0.0.6-py3-none-any.whl --force-reinstall
```
